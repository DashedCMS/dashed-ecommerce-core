# Betaalmethodes

Beheer welke betaalmethodes beschikbaar zijn voor je klanten. Ga naar **Instellingen > Betaalmethodes**.

## Beschikbare betalingsproviders

DashedCMS ondersteunt meerdere betalingsproviders:
- **Mollie** - iDEAL, creditcard, Bancontact, PayPal, etc.
- **Pay.nl** - Diverse betaalmethodes
- **MultiSafepay** - Diverse betaalmethodes

## Betaalmethode instellen

Per betaalmethode kun je instellen:
- **Naam** - De naam die klanten zien
- **Extra kosten** - Eventuele toeslag voor deze betaalmethode
- **Minimaal bedrag** - Minimum bestelbedrag voor deze methode
- **Actief** - Schakel de methode in/uit

## API koppeling

Ga naar de instellingen van je betalingsprovider om de API-sleutels in te voeren. Na het invoeren worden de beschikbare betaalmethodes automatisch opgehaald.

**Gevolg:** Klanten zien alleen de betaalmethodes die je hebt geactiveerd bij de checkout.

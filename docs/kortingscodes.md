# Kortingscodes

Ga naar **E-commerce > Kortingscodes** om kortingscodes aan te maken en te beheren.

## Een kortingscode aanmaken

1. Klik op **"Nieuw"**
2. Vul de verplichte velden in:
   - **Naam** - Interne naam (niet zichtbaar voor klanten)
   - **Code** - De code die klanten invoeren (bijv. "KORTING10")
   - **Type** - Vast bedrag of percentage
   - **Waarde** - Het kortingsbedrag of percentage

## Opties

### Geldigheidsperiode
Stel een start- en einddatum in. De code is alleen geldig binnen deze periode.

### Voorraad / Gebruik limiet
Beperk hoeveel keer de code gebruikt mag worden. Elke keer dat de code wordt toegepast, wordt de voorraad met 1 verlaagd.

### Eenmalig per klant
Als dit aan staat, kan elke klant de code maximaal 1 keer gebruiken.

### Minimaal bestelbedrag
Stel een minimumbedrag in dat in de winkelwagen moet zitten voordat de code geldig is.

### Site beperking
Als je meerdere sites hebt, kun je de code beperken tot specifieke sites.

## Automatische kortingscodes

Kortingscodes worden ook automatisch gegenereerd door het verlaten winkelwagen systeem. Deze codes hebben een prefix (standaard "TERUG") en zijn eenmalig bruikbaar.

# Facturen

Facturen worden automatisch gegenereerd bij betaalde bestellingen.

## Automatische facturatie

Bij elke betaalde bestelling wordt automatisch een factuur aangemaakt met een oplopend factuurnummer. De factuur bevat:
- Bedrijfsgegevens
- Klantgegevens
- Bestelde producten met prijzen
- BTW specificatie
- Totaalbedrag

## Factuur instellingen

Ga naar **Instellingen > Facturatie** om de factuurinstellingen aan te passen:

- **Factuurnummer prefix** - Voorvoegsel voor factuurnummers (bijv. "F-")
- **Volgend factuurnummer** - Het volgende factuurnummer dat wordt gebruikt
- **Bedrijfsgegevens** - Naam, adres, KvK-nummer, BTW-nummer
- **Logo** - Het logo op de factuur
- **Voettekst** - Tekst onderaan de factuur (bijv. betaalgegevens, voorwaarden)

## Factuur downloaden

Facturen zijn beschikbaar als PDF:
- In het admin panel bij de bestelling
- Via de e-mail die de klant ontvangt
- Via het klantaccount op de website
